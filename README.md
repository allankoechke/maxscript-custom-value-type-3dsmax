# maxscript-custom-value-type-3dsmax

> Making custom value types to be exposed to MAXScript using the 3ds C++ SDK  
> Project described in the [blog](https://blog.autodesk.io/custom-valuetype-maxscript/)

Developed for 3ds Max 2023, for newer versions, project may need some minor readjustments to build.

> Create a ticket through the ADN portal if encounter any issues building/running the sample.
